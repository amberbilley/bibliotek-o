Biblioteko Ontology Site

This site biblioteko.org hosts the Biblioteko ontology. The ontology is available in HTML and RDF/XML via content negotiation from http://biblioteko.org/ontology/ or from http://biblioteko.org/ontology.rdf.

This ontology is a product of the Linked Data for Libraries (LD4L) Labs and Linked Data for Production (LD4P) projects.
