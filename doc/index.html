bibliotek-o: a BIBFRAME 2 Extension Ontology

bibliotek-o is an ontology recommendation for modeling bibliographic metadata. It includes:

  • The bibliotek-o extension ontology, which defines additions and modifications to BIBFRAME 2 and is intended to be used as a supplement to the core BIBFRAME 2 ontology.
  • A set of ontology fragments (including BIBFRAME 2) intended for use with the bibliotek-o extension.
  • An application profile - i.e., a set of models and patterns documenting the recommended use of the bibliotek-o and external ontologies.

This site hosts the bibliotek-o ontology. The ontology is available in HTML and RDF/XML OWL via content negotiation from http://bibliotek-o.org/ontology/ or from http://bibliotek-o.org/ontology.owl.
