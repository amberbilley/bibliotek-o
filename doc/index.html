biblioteko: a BIBFRAME 2 Extension Ontology

This site bibliotek-o.org hosts the biblioteko ontology. The ontology is available in HTML and RDF/XML OWL via content negotiation from http://bibliotek-o.org/ontology/ or from http://bibliotek-o.org/ontology.owl.

The Biblioteko ontology is a product of the Linked Data for Libraries (LD4L) Labs and Linked Data for Production (LD4P) projects.
